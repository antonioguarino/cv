Projects


Comau SmartSiX and MTB SCARA Pick&Place task

The project regards th planning and implementation of the end-effector motion to accomplish a pick&place task. The project involved two robots: a dummy one, the MTB SCARA (which has been modeled in VREP) and the real SmartSiX robot produced by COMAU Robotics. The produced software includes the following features:

  • Direct kinematics calculus;
  • Trajectory planning through rect, arc and trapezoidal profile;
  • Kinematic inversion through second order CLIK with Jacobian inverse and null projection for obstacle avoidance.

Dynamic model identification for the Comau SmartSiX robot

The produced software implements the dynamic model identification technique discussed in [1]. The implementation requires the minimization of the objective function showed in the referenced paper and the generation of the exciting trajectory. The obtained trajectory has been executed on the robot and, starting from the motors' currents, it has been possible to calculate the dynamic model of the robot.

Robot Control schemes simulation

The produced software simulates different control schemes for an industrial manipulator in order to give the possibility to compare the control result in terms of tracking error. The supported control schemes are:

  • Independent joint control
  • PD with gravity compensation
  • Inverse dynamic (Joint Space)
  • Inverse dynamic (Operative Space)

Perturb&Observe MPPT algorithm implementation for a photovoltaic battery charger

A photovoltaic battery charger simulation has been implemented and the Perturb and Observe [2] (P&O) algorithm for the Maximum Power Point Tracking (MPPT) has been set up for the proposed scenario. In particular the P&O algorithm has been coded in order to be ready for FPGA deployment. The board considered for the project is the Zybo board by Xilinx.

I-V curve reconstruction algorithm for a photovoltaic string in non-homogeneous conditions

Getting the I-V curve of a photovoltaic (PV) field is essential to give stiffness to the underlying control techniques, estimate the energy production of the plant or to give support to fault diagnosis techniques. The implemented software allows the reconstruction of the whole curve starting from the number of pv panels in series and the working conditions (irradiance and temperature). The software has been built giving particular attention to numerical optimization as showed in [3] and [4], and code optimization for FPGA. The final software version has been synthesized, deployed and tested on Xilinx Zybo board.

Design and simulation of a digital voltage mode controller for a Wireless Battery Charger

Project goal is to design the control of a buck dc-dc converter for a wireless battery charger. The control has been designed and analyzed in analog (using the k-factor method [5] ) and then it has been discretized. After the discretization the major project choices about the digital implementation have been done (minimum cpu clock requirements, minimum adc requirements, limit cycle avoidance techniques etc). The resulting design has been simulated using PSIM by powersimtech.

Inductor current and voltage acquisition of a DC-DC buck converter

Sensing the inductor current and voltage is a key element to understand the operating conditions of a dc-dc power converter. The project goal is to implement the data acquisition system for those two signal [6] and deploy the software on the TMS320F280x MCU produced by TI. The acquired data are postprocessed and stored on board for further analysis.
